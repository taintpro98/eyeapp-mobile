The same product and the same warm-purple design language as the web app — VN-Index KPIs, trading signals, watchlist, portfolio — carried into a phone. Two directions below. Variation 1 stays faithful to the web (iOS, light theme); Variation 2 is a more native, elevated take (Android, dark theme). Every screen shares the same tokens — #6F2C91 brand, warm #F4F0EA surfaces, Inter, 12–14px cards — so either theme works on either platform.
